# Student Management System

A mini project built in C, focused on applying core data structures and algorithms to manage student records through a simple console-based application.

# About

This project simulates a basic Student Management System where users can add, update, delete, and search student records, along with performing simple analytics like finding the highest and lowest scorers. It was built as a hands-on exercise in applying arrays, structures, and control flow logic in C.

# Features

- Add new student records
- Update existing student details
- Delete student records
- Search for a specific student
- Display all student records
- Find the highest scorer
- Find the lowest scorer
- Simple menu-driven console interface

# Tech Stack

- **Language:** C
- **Concepts used:** Structures, Arrays, Functions, Loops, Conditional Logic

# How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Badal-devlop/Student-management-system.git
   cd Student-management-system
   ```

2. Compile the code:
   ```bash
   gcc student_management.c -o student_management
   ```

3. Run the executable:
   ```bash
   ./student_management
   ```
